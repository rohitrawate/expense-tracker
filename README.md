# Expense Tracker

An enterprise-grade Expense Tracker built with Java 21 and Spring Boot 4.1.0.

This project is being developed from the ground up following production-ready software engineering practices, including clean architecture, RESTful APIs, Spring Security, PostgreSQL, Docker, testing, CI/CD, monitoring, and cloud deployment.

The objective is not only to build a complete application but also to demonstrate how modern backend systems are designed, implemented, tested, and maintained in enterprise environments.

## Project Goals

- Learn Spring Boot from fundamentals to enterprise development.
- Build a production-ready Expense Tracker.
- Apply software engineering best practices.
- Maintain a clean Git history.
- Prepare for SDE-1 and SDE-2 backend interviews.

## Getting Started

### Prerequisites

- Java 21
- Maven
- Git

### Clone

git clone https://github.com/rohitrawate/expense-tracker.git

### Run

./mvnw spring-boot:run
